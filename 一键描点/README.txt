
一键描点
By FlyingSky

====================

使用方法：
丢到UTAU的plugins文件夹即可

====================

热键设置：
双击运行WinHotKey.exe
再双击运行WinHotKeyCfg.exe
在打开的窗口中单击New Hotkey..
在弹出的对话框中：
  Description of this hotkey  设置热键名称（随意）
  When I press these keys & Along with the key  设置热键（随意）
  I want WinHotKey to  默认不改（Launch an Application）
  Launch which application：
    单击Browse
    在弹出的对话框中找到本插件文件夹下的hotkey.vbs并双击
    单击OK
设置完成
PS：若有疑问可前往FlyingSky's Blog(http://blog.flyingsky.ml/)察看图文教程
PSS：请确认通知栏中有WinHotKey图标
